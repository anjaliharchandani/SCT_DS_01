# 📊 Data Science Task 1 - Population Analysis using Tableau

## ✅ Task Overview

This project focuses on **analyzing and visualizing population data** using Tableau. The primary objective is to highlight the **Top 10 most populous countries in 2022** through a clean and effective bar chart.

---

## 📌 Objective

- Import population dataset into Tableau
- Create a bar chart showing the most populated countries
- Highlight trends and differences visually
- Publish and share the visualization

---
